This is a demo of a Google Plus Stream.
This was introduced by Roman Nurik in the Google IO 2012 app.

I just copy/pasted his code for easier share.
Please thank him on Google plus: https://plus.google.com/113735310430199015092/posts

My Google Plus page: https://plus.google.com/108315424589085456181

HOW TO USE:

- Modify Private.java and add your api key (https://developers.google.com/accounts/docs/OAuth2#Registering)
- Add the awesome ActionBarSherlock library: http://actionbarsherlock.com/
- Customize the keyword for the post to display in  SocialStreamFragment.java

Enjoy!
